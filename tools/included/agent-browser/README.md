# Agent Browser

`agent-browser` is the shipped browser automation tool for `vercel-claw`.

Use it when the model needs a persistent browser session for:

- opening pages
- reading titles and accessibility snapshots
- clicking refs and filling inputs
- capturing screenshots inside the sandbox workspace

Start with `knowledge/about.md` for the short product summary, then inspect `mcp.json` and the executable operations in `index.ts` if you need the exact contract.
